# data/

This directory contains:

- [bus-history-44.csv](bus-history-44.csv): Historical data on the predicted and 
  actual arrival times for a single bus (the #44 in King County) to use as an
  example for fitting a probabilistic model of time to arrival. 
- [known_p.csv](known_p.csv): The correct answers to probabilistic questions 
  in the survey, generated from the same distributions used to generate the plots
  in the survey.
- [survey.csv](survey.csv): Answers to the survey questions
